# Editor

## Vi, Vim
- Normal mode (Esc):
- Edit mode (i, a):
- Command (:):
	- w: save
	- q: quit
	- yy, 5yy: copy 5 lines;
	- dd, 5dd: delete 5 lines;
	- u: undo;
	- /: find, n for next;
	- set nu; set nonu: set line number;
	- G: last line;
	- gg: first line;
	- 8 shift-g, 8 gg: go to 8-th line;
	- Ctrl + f: front 1 page;
	- Ctrl + b: back 1 page;
	- Ctrl + d: front half page;
	- Ctrl + u: back half page;
